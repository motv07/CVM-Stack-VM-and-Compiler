# CVM++
### Stack-Based Virtual Machine & Custom Compiler

CVM++ is a custom scripting language built entirely in **C++**, designed to demonstrate how programming languages work internally.

The project implements the complete language execution pipeline:

```text
Source Code
    в†“
Lexer
    в†“
Parser
    в†“
Abstract Syntax Tree (AST)
    в†“
Bytecode Compiler
    в†“
Stack-Based Virtual Machine
    в†“
Execution Output
```

Instead of relying on existing runtimes, CVM++ compiles source programs into proprietary bytecode executed by a custom stack-based Virtual Machine.

---

# Features

- Lexer & Tokenizer
- Recursive Descent Parser
- Abstract Syntax Tree (AST) with debug printer
- Bytecode Compiler
- Stack-Based Virtual Machine
- **Lexical Block Scoping (Environment Stack)**
- Variables: Declaration (`let`) and Reassignment
- Arithmetic Operations:
  - `+`
  - `-`
  - `*`
  - `/`
- Logical & Comparison Operators:
  - `==`
  - `>`
  - `<`
  - `&&`
  - `||`
- Boolean values:
  - `1 в†’ true`
  - `0 в†’ false`
- Nested `if / else`
- `while` loops
- `print()` built-in
- `input()` built-in
- REPL (Read-Eval-Print Loop)
- File Runner
- Debug mode:
  - AST printing
  - Bytecode tracing
  - VM execution trace
- Runtime error handling
- Division-by-zero prevention

# Language Syntax

## Variable Declaration

```c
let x = 10;
let y = input();
```

---

## Arithmetic

```c
print(a + b);
print(a - b);
print(a * b);
print(a / b);
```

---

## Logical Operators

```c
print(a && b);
print(a || b);
```

---

## Comparisons

```c
print(x > 0);
print(x < 0);
print(x == 0);
```

---

## Block Scoping

```c
let x = 10;

if (x > 0)
{
    let y = 20;
    print(y);
}

print(x);
```

Variables declared inside blocks exist only within their lexical scope.

---

## if / else

```c
if (x > 0)
{
    print(1);
}
else
{
    print(0);
}
```

Nested conditions are supported.

---

## while Loop

```c
let x = input();

while (x > 0)
{
    print(x);
    x = x - 1;
}
```

# Opcode Instruction Set (ISA)

| Opcode | Description |
|---------|------------|
| PUSH | Push constant value onto stack |
| ADD | Pop two values and push sum |
| SUB | Pop two values and push difference |
| MUL | Pop two values and push product |
| DIV | Pop two values and push quotient (zero-check enabled) |
| AND | Pop two values and push logical AND |
| OR | Pop two values and push logical OR |
| DEFINE | Initialize variable in current scope |
| STORE | Update existing variable |
| LOAD | Load variable value |
| ENTER_SCOPE | Create new lexical environment |
| EXIT_SCOPE | Destroy current environment |
| GT | Greater-than comparison |
| LT | Less-than comparison |
| EQ | Equality comparison |
| PRINT | Output stack top |
| INPUT | Read integer input |
| JMP | Unconditional jump |
| JMP_IF_FALSE | Conditional jump |
| HALT | Stop execution |

# Grammar

```text
program     -> statement* EOF

statement   -> varDecl
             | assignment
             | printStmt
             | ifStmt
             | whileStmt
             | block

varDecl     -> "let" IDENTIFIER "=" expression ";"

assignment  -> IDENTIFIER "=" expression ";"

printStmt   -> "print" "(" expression ")" ";"

ifStmt      -> "if" "(" expression ")" block
               ("else" block)?

whileStmt   -> "while" "(" expression ")" block

block       -> "{" statement* "}"

expression  -> logical

logical     -> equality
               (("&&" | "||") equality)*

equality    -> comparison
               ("==" comparison)*

comparison  -> term
               ((">" | "<") term)*

term        -> factor
               (("+" | "-") factor)*

factor      -> unary
               (("*" | "/") unary)

unary       -> "-" unary
             | primary

primary     -> NUMBER
             | IDENTIFIER
             | "(" expression ")"
             | input()
```
